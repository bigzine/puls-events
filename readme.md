# Puls-Events — Assistant RAG pour événements culturels (POC)

## Objectif du projet
POC démontrant la faisabilité d'un chatbot capable de répondre à des questions
sur des événements culturels à venir, en s'appuyant sur un système RAG
(Retrieval-Augmented Generation) combinant recherche vectorielle (FAISS) et
génération de réponse en langage naturel (Mistral, via LangChain).

- **Zone géographique ciblée** : Paris & Île-de-France
- **Source de données** : API Open Agenda
- **Fenêtre temporelle** : événements des 12 derniers mois
